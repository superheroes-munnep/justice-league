# How to 
- clone to local repository
```
git clone https://github.com/munnep/justice-league.git
```
- go to directory
```
cd justice-league
```
- switch to branch feature-release
```
git checkout feature-release
```
- add the changes
```
git add .
```
- commit the changes
```
git commit -am "changes made"
```
- push the changes
```
git push 
```
- changes should be visable in the UI
